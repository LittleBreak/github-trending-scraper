# GitHub Trending 小红书文案生成

基于 [GitHub Trending](../output/current_trending.json) 数据生成小红书分享文案。

## 输出位置

```
output/post.md
```

## 文案结构

| 序号 | 模块 | 说明 |
|------|------|------|
| 1 | 标题 | 一个有概括性、吸引人的主标题 |
| 2 | 项目列表 | Top10 项目列表（排名、项目名、语言、Stars、简介） |
| 3 | 趋势总结 | 3-4 条精炼的技术趋势洞察，分析最近的技术发展和趋势 |
| 4 | 互动引导 | 点赞、关注引导语 |
| 5 | 内容标签 | 相关话题标签，提升曝光（如 #github #开源 #AI） |

## 格式要求

小红书不支持 Markdown 语法，生成的文案必须是**纯文本风格**：

| 元素 | 禁止使用 | 替代方案 |
|------|----------|----------|
| 标题 | `#` `##` | 直接写文字，无需标记 |
| 表格 | `\| \| \|` | 列表形式，每项独立成行 |
| 分隔线 | `---` | `━━━━━━━━━━` |
| 强调 | `**加粗**` | emoji 图标引导（如 💡） |
| 列表 | `- item` | 数字 emoji（①②③）或符号 |
| 排名 | 纯数字 | 奖牌 emoji（🥇🥈🥉）+ 数字 emoji（4️⃣5️⃣...） |

## 关键发现（来自数据分析）

- AI Coding Agent 主导榜单（6/10 与 Claude Code 生态相关）
- 核心项目：opencode, vibe-kanban, superpowers, skills, claude-code
- 字节跳动 UI-TARS 入榜
- memos 代表自托管/隐私优先趋势
