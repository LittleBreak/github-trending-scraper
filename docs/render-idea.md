将每个项目的内容以卡片的形式渲染出来，每张卡片包含项目名称、Star 数量、技术栈等信息。并以图片的形式保存下来。

## 方案
为了让图片在小红书上好看，强烈建议不要直接用代码“画”图（PIL），而是用 HTML/CSS 写好样式，然后截图。这样排版最灵活，设计上限最高。

技术栈: Jinja2 (HTML模板引擎) + Playwright (浏览器自动化截图)。

实现逻辑:

设计模板: 写一个 HTML 文件，设计一个精美的卡片样式（参考小红书流行的 INS 风、极简风）。使用 Flexbox/Grid 布局。

数据填充: 使用 Jinja2 将爬取到的 Top 10 数据填入 HTML 模板中。

截图保存:

启动 Playwright 无头浏览器。

加载生成的 HTML。

对每个卡片的 DOM 元素（比如 <div class="card">）进行截图，保存为 top1.png, top2.png 等。
